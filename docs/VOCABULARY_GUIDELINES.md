## Vocabulary Audit Process

1. **Review each sentence** for vocabulary level appropriateness
2. **Replace advanced words** with simpler alternatives
3. **Maintain meaning** while using tier-appropriate vocabulary
4. **Test with target learners** to ensure comprehension

## Quality Assurance

- **Tier 1**: Should be comprehensible to complete beginners
- **Progressive difficulty**: Each tier introduces slightly more complex vocabulary
- **Cultural relevance**: Use vocabulary relevant to learners' needs
- **Frequency-based**: Prioritize high-frequency words in lower tiers
